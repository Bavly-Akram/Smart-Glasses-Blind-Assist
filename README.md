This project is a wearable device that helps visually impaired people detect nearby obstacles.

##  How It Works
- Uses an **ultrasonic sensor** to measure distance
- When an object or wall is detected within a safe range, a **buzzer/speaker** alerts the user
- Built using **Arduino Uno** and powered by a **9V battery**
- All components are mounted on glasses for easy and hands-free use

##  Components Used
- Arduino Uno  
- Ultrasonic Sensor (HC-SR04)  
- Buzzer / Speaker  
- 9V Battery  
- Wires & Glasses frame  

##  Team
This project was developed by a team of five college students.
